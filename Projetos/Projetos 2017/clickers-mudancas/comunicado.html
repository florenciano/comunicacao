Caros professores,

o software TurningPoint, utilizado para votação eletrônica com os Clickers, foi atualizado nos computadores do Insper para uma nova versão.

O que mudou?

Agora, assim que o software é aberto, é necessário realizar o login com usuário e senha cadastrados. Na versão anterior, o software só exigia o login no momento de gerar o número da sessão.

As demais funcionalidades não sofreram alteração e, para os alunos, não houve mudanças.

Pedimos que atualizem o software nos computadores pessoais com a nova versão que está disponível aqui.

Caso possua apresentações que já utilizem o Responseware, é recomendado que faça um breve teste nas mesmas, a fim de verificar se os slides não sofrem nenhuma desconfiguração.

Em caso de dúvidas, ou caso precise de auxílio, estamos à disposição.
