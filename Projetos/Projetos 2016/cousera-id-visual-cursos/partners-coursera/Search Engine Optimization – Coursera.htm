Search Engine Optimization

Search engine optimization (SEO) is the practice of improving and promoting a website in order to increase the number of visitors the site receives from search engines.

Titles, URLs, and on-page content are the critical elements that search engines consider when displaying a course or Specialization as a match for particular search query. Because most new users will discover courses via a web search, following the SEO guidelines below can dramatically increase the course’s visibility to potential learners.

SEO Best Practices

Course and Specialization titles

  • Include 1-3 popular, relevant industry keywords in a course or Specialization title. To identify strong keywords, compare candidate terms using Google Trends.
  • Terms used in the title should be as concise and straightforward as possible, without sacrificing accurate descriptions of the course or Specialization content.
  • Place keywords at the beginning of a title. Example: For the keyword machine learning, “Machine Learning Fundamentals” is much better than “Theoretical Frameworks and Practical Applications of Machine Learning” because it places machine learning first and is more concise.

Course and Specialization URLs

  • Create URLs using 1 - 3 keywords from the title. Use full words and replace-spaces-with-hyphens. 
  • Avoid unnecessary numbers or on-campus jargon. For example, machine-learning-basics is better than ml-101 because it uses full, human-readable words.
  • Create URLs with descriptive words that match words in your course title, such as machine-learning-basics, machine-learning-algorithms, etc.
  • When creating a Specialization, avoid creating repetitive URLs for individual courses, such as machine-learning-1, machine-learning-2, etc. 
  • Note: You cannot change URLs after a course is published. Please double-check this step before committing and publishing.

Page Content

Include important keywords and close variants in course description and module titles, where possible.

Selecting SEO Keywords

Selecting the right keywords is key to improving discoverability. To get started, generate a list of ideas by completing the following statements:

  • “When a user searches for the word or phrase ____ on Google, they should see my course.” 
  • “When I think of my course, the first words that come to mind are ____.” 
  • “If my course was in a Wikipedia article, the Wikipedia title would be ____.” 
  • “When I search for ____ on Google, I think my course is a better result than what I see today.”

Then use Google Trends to compare terms for relative search popularity. Google Trends can compare up to five keywords and generate a graph of relative monthly search volume.

Case Study: Search Engine Optimization

Let’s imagine you are creating a Specialization for making ice cream for Vermont State University (VSU). You’re not sure if you should go with “Ice Cream Making,” “Frozen Dessert Design” or “Ice Cream Design” as titles. You’re also not sure how to choose URLs to optimize search traffic.

Here is how we would think through the process of identifying keywords, doing research, and creating optimized titles.

1) Did we choose popular and relevant keywords?

Here’s a sample query from Google Trends; “Ice Cream Making” sees the highest search volume. When I search “ice cream making” in Google, I see results highly relevant to my learning material.

 

2) Did we create a Specialization URL that uses the keywords in our title?

  • Good examples: ice-cream, ice-cream-making, making-ice-cream
  • Bad examples:
    • ic, icm: Uses abbreviations which are confusing and not commonly used.
    • dessert-making, ice-cream-churning: Does not use keywords from title.
    • icecreammaking: Needs hyphens. Search engines read this as one single word: “icecreammaking”.
    • ice-cream-VSU: Although this includes the school’s name, this is not a common phrase and dilutes the value of the ice-cream term.

3) Which course URLs optimize for search?

  • Good examples: ben-and-jerrys, ice-cream-flavors, ben-jerrys-flavors (Note: You can exclude articles like “and”, “or”, “the”, etc.)
  • Bad examples:
    • ice-cream-making-1: We already used this term in the Specialization URL; there is no need to rank again on these terms.
    • phish-food, cherry-garcia: Relevant terms, but not used in course title.
    • bjicf, ben-jer-fla: Uses confusing, not commonly used abbreviations.
